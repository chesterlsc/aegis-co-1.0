# Aegis & Co 1.0

First production design iteration for the Aegis & Co website.

Aegis & Co is positioned as a systems company: calm, precise, modern, operational, and focused on helping businesses run with less drag, better control, and clearer visibility.

## Files

- `index.html` - Main static site markup.
- `styles.css` - Complete visual system, responsive layout, diagrams, and subtle SVG line animations.

## Preview

Open `index.html` in a browser, or run a local static server from this folder:

```sh
python3 -m http.server 4173 --bind 127.0.0.1
```

Then visit:

```text
http://127.0.0.1:4173/index.html
```
